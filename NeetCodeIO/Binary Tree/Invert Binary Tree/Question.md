### Invert Binary Tree

#### Problem Description

You are given the root of a binary tree `root`. Invert the binary tree and return its `root`.

**_Example 1:_**
**Input:** root = [1,2,3,4,5,6,7]
**Output:** [1,3,2,7,6,5,4]

**_Example 2:_**
**Input:** root = [3,2,1]
**Output:** [3,1,2]

**_Example 3:_**
**Input:** root = []
**Output:** []

**_Constraints:_**

- `0 <= The number of nodes in the tree <= 100.`
- `-100 <= Node.val <= 100`
