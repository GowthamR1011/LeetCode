### Reverse Linked List

#### Problem Description

Given the beginning of a singly linked list `head`, reverse the list, and return the new beginning of the list.

**_Example 1:_**
**Input:** head = [0,1,2,3]
**Output:** [3,2,1,0]

**_Example 2:_**
**Input:** head = []
**Output:** []

**_Constraints:_**

- `0 <= The length of the list <= 1000.`
- `-1000 <= Node.val <= 1000`
