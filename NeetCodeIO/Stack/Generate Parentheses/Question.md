### Generate Parentheses

#### Problem Description

You are given an integer `n`. Return all well-formed parentheses strings that you can generate with `n` pairs of parentheses.

**_Example 1:_**
**Input:** n = 1
**Output:** ["()"]

**_Example 2:_**
**Input:** n = 3
**Output:** ["((()))","(()())","(())()","()(())","()()()"]
You may return the answer in any order.

**_Constraints:_**

- `1 <= n <= 7`
