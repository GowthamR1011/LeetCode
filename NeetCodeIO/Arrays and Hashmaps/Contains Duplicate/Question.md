### Contains Duplicate

#### Problem Description

Given an integer array `nums`, return `true` if any value appears more than once in the array, otherwise return `false`.

**_Example 1:_**
**Input:** nums = [1, 2, 3, 3]
**Output:** true

**_Example 2:_**
**Input:** nums = [1, 2, 3, 4]
**Output:** false
